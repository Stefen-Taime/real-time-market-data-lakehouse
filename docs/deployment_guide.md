# Deployment Guide

## Prerequis

- acces au workspace Databricks cible
- variables et secrets renseignes
- configuration d'environnement completee

## Etapes

1. valider le bundle Databricks
2. deployer les ressources sur l'environnement cible
3. executer les jobs d'initialisation si necessaire
4. verifier dashboards, alertes et jobs apres deploiement
